# Agentic Automata Learning — project page

Static site for *Can LLM Agents Infer World Models? Evidence from Agentic Automata Learning*.

## Deploy to GitHub Pages
1. Put `index.html` and the `static/` folder in a repo.
2. Settings → Pages → Source: `main` branch, `/ (root)`.
3. Site serves at `https://<user>.github.io/<repo>/`.

## Files
- `index.html` — the page. Figures are embedded; the only external asset is the demo video.
- `static/demo.mp4` — demo screencast shown in the first tab (autoplays muted, loops).

A single-file alternative (`index.html` with the video embedded too) is provided separately if you want zero assets.
