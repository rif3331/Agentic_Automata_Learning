# Agentic Automata Learning

**Paper • Web Interface • Source Code**

Agentic Automata Learning is an evaluation framework for studying and
benchmarking Large Language Model (LLM) agents. The framework
investigates whether agents can infer a hidden model of an environment
through interaction, information gathering, and iterative hypothesis
refinement.

The project is based on the classic **Active Automata Learning**
setting, where an agent must uncover a hidden Deterministic Finite
Automaton (DFA) by interacting with an oracle through queries. This
formulation provides a controlled evaluation environment with scalable
task complexity, formal performance metrics, and direct comparisons to
classical learning algorithms.

The project includes a research paper describing the methodology and
experimental findings, an interactive web interface for running
experiments and analyzing results, and the full source code of the
evaluation framework.

## Components

-   📄 **Research Paper** -- Introduces the Agentic Automata Learning
    framework and presents experimental results on modern LLM agents.
-   🌐 **Web Interface** -- Agentic Automata Learning Runner, an
    interactive interface for launching experiments and visualizing
    agent behavior.
-   💻 **Source Code** -- Complete implementation of the evaluation
    framework, benchmarks, analysis tools, and experiment runner.
